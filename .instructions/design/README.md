# AxodusAPP Nucleus Design Instructions

This directory stores product-specific design instructions for each AxodusAPP nucleus.

The root app design system remains centralized in:

- `.instructions/DESIGN.md`
- `.instructions/UI_UX.md`
- `.instructions/ARCHITECTURE.md`

Each nucleus may add its own:

- information architecture
- layout priorities
- component direction
- density rules
- workflow emphasis
- domain-specific UX constraints

Nucleus design files must preserve the shared AxodusAPP design system and token architecture.
